Le dossier 'MEDIA' present dans le dossier 'OS' est le dossier média d'origine.

Le fichier OS.LST contient la liste des systèmes d'exploitations présents.
	La syntaxe est très simple, il s'agit du nom de l'OS et le nom de dossier source
	séparée par ':'
	Par exemple, j'ai un OS CraftyOS dans le dossier 'CRAFTYOS', j'ajouterai a la liste
	« CraftyOS:CRAFTYOS »
	Le premier de la liste sera la premier si la commande "Demarrer/" est executée
	sans preciser le nom de l'OS ;-)


BON A SAVOIR :
	Si vous voulez partir depuis ZERO :
	- Supprimez le dossier MyOS, WacOS vider le fichier OS.LST.
	- Creer votre dossier pour votre OS et remplissez le fichier OS.LST.
	- Copiez le dossier MEDIA et le profil graphique DANS votre dossier d'OS et modifiez ceci.
